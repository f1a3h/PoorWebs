## PoorWebs

这是一个计算机基础课的大作业。

要求使用 HTML 语言编写 5 ~ 10 个网页，可以通过链接跳转。

顺便熟练一下 git 的操作，我就传到 Github 上了。

因为没什么狠活，而且也是 HTML 新手 + 菜鸟，所以只是瞎搞了一个简陋到不能再简陋的博客（雾），为了凑足网页数量，蒯了 [Let Me Baidu That For You](https://github.com/bangbang93/lmbtfy.cn) 这个开源项目的代码。

本项目一共包含 5 个页面。
